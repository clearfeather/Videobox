# Changelog

## v1.0.1

- Added optional PayPal donation and GitHub Sponsors links
- Removed unused music and pictures library capabilities from the app package

## v1.0.0

Initial VideoBox release from the ClearFeather fork of Screenbox.

- Renamed the visible app experience to VideoBox
- Added folder-first video browsing and an All Videos view
- Added favorites, recent media, video tags, and tag navigation
- Added folder covers and custom video thumbnail options
- Added optional automatic thumbnail scanning and thumbnail cache controls
- Added filename cleanup for more presentable library titles
- Added metadata details for selected videos
- Added local app settings for startup page, volume behavior, and PIN lock
- Removed visible music and network browsing from the customized navigation
