# VideoBox Privacy Policy

VideoBox is a local-only media player forked from Screenbox.

## Local media

VideoBox reads the folders and media files you choose to add to the app. Library settings, tags, favorites, recent playback history, thumbnail cache data, selected custom thumbnails, and playback preferences are stored locally in the app's Windows application data folder.

VideoBox does not upload your media library, filenames, tags, watch history, thumbnails, or playback activity.

## Network and telemetry

This fork is intended to run without internet access and does not include analytics or crash telemetry for VideoBox.

Some upstream Screenbox code, dependency names, or documentation may still appear in the source tree for attribution and compatibility, but this fork should not send VideoBox usage data to ClearFeather, Screenbox, or the original author.
